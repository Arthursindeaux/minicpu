## Tema

**Média de Inteiros**

O objetivo desta atividade foi simular o ciclo Fetch → Decode → Execute de uma CPU e resolver o problema proposto utilizando a arquitetura MiniCPU.

O programa calcula a média de valores predefinidos armazenados na memória e salva o resultado no endereço especificado.
